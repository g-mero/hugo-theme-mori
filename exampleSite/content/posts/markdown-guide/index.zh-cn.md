本文介绍常用的 Markdown 语法。

## 标题

使用 `#` 创建标题，数量代表级别。

## 文本样式

- **粗体文本**
- _斜体文本_
- ~~删除线~~

## 链接和图片

[这是一个链接](https://example.com)

## 代码

行内代码：`const x = 1;`

代码块：

```python
def hello_world():
    print("Hello, World!")
```

## 列表

无序列表：

- 项目 1
- 项目 2
- 项目 3

有序列表：

1. 第一步
2. 第二步
3. 第三步

## 引用

> 这是一段引用文字。
> 可以有多行。

## 表格

| 列 1 | 列 2 | 列 3 |
| ---- | ---- | ---- |
| A    | B    | C    |
| D    | E    | F    |

这就是基本的 Markdown 语法！
